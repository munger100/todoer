This is a simple todo app, that aims at being useful yet minimalistic. It also serves as a POC for certain technologies.

Technologies used:
  - NextJS, running on NodeJS
  - Typescript

Technologies planned to be used:
  - Docker
  - Prisma

First, to run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
